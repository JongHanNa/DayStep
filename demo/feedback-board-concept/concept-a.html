시안 A의 안티패턴 해결 방식

✅ 정보 과부화 해결

타인 게시물을 리스트에서 완전히 배제. 커뮤니티 활성도는 상단 집계 한 줄("48건 제보 중")로만 암시 → 화면에 오직 "내 것"만. 인지 부담 최소.

✅ 불명확한 시작점 해결

화면에서 유일하게 primaryColor로 채워진 요소는 우하단 FAB 하나. 나머지 UI는 모두 중립색. 시선이 자동으로 "새 제보하기"로 수렴.

✅ 플로우 단절 해결

작성 → 시트 → 제출 → 리스트 갱신까지 화면 전환 0회. `.sheet` 닫히며 카드 리스트 상단에 신규 카드 스프링으로 삽입되는 물리감 유지.

✅ 시각적 계층

34 → 22 → 17 → 13 → 11 의 5단계 타이포 스케일. 색상은 status pill 3곳 + FAB 1곳 총 4곳에만 허용. 나머지는 순수 neutral.

🔒 프라이버시 모델

RLS: auth.uid() = user_id OR users.role = 'admin'. 타인 게시물은 서버에서부터 필터링되어 이 화면에 도달 불가능.
