# Patterns (패턴 라이브러리)

## 📁 이 폴더의 용도

프로젝트에서 사용하거나 사용했던 코드 패턴들을 보관합니다.

**왜 보관하나요?**
- 나중에 다시 사용할 수 있는 패턴 참고
- 여러 구현 방식 비교 및 선택
- 코드 재사용성 향상

## 📦 현재 보관 중인 패턴

### modals/ - 모달 패턴

#### 1. DaisyUIModal
DaisyUI의 `<dialog>` 컴포넌트 기반 모달

**예제:** [modals/DaisyUIModal/example.tsx](./modals/DaisyUIModal/example.tsx)

#### 2. ReactModalSheet
react-modal-sheet 라이브러리 기반 모달

**예제:** [modals/ReactModalSheet/example.tsx](./modals/ReactModalSheet/example.tsx)

---

## 📚 사용 방법

1. 필요한 패턴의 `example.tsx` 파일 열기
2. 코드 복사 후 프로젝트에 맞게 수정
3. 패턴 선택 기준은 `CLAUDE.local.md` 참고

---

## 🔮 향후 추가 예정

이 폴더에는 다음과 같은 카테고리의 패턴들도 추가될 수 있습니다:
- `components/` - 컴포넌트 패턴 (폼, 리스트, 카드 등)
- `hooks/` - 커스텀 훅 패턴
- `state-management/` - 상태 관리 패턴
- `layouts/` - 레이아웃 패턴
