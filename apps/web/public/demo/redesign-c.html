✨ 인터랙션 데모

상단 프로젝트 요약 카드를 클릭하면, 해당 프로젝트 할일만 하이라이트됩니다:

디자인 노트

  • + "오늘의 프로젝트" 요약 카드 → 프로젝트 중심 관리 가능
  • + 프로젝트별 완료율 진행 링 → 동기부여 피드백 극대화
  • + 사이드바 + 배지 + 커넥터 → 시간축 위의 프로젝트 블록 시각화
  • + 프로젝트 카드 클릭 → 필터/하이라이트 → 빠른 포커스 전환
  • + ADHD 사용자: "왜 이걸 해야 하지?" 단서 풍부하게 제공
  • ~ UI 복잡도 증가 → 정보 과부하 가능성
  • ~ 프로젝트가 많을 때 요약 카드 영역이 커질 수 있음
  • - 구현 비용 가장 높음 — 요약 카드, 커넥터, 하이라이트 로직
  • - 프로젝트 요약 데이터 집계 로직 필요 (전체 할일 수, 완료 수)
