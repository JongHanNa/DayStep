A: 미니멀 리스트 iOS Settings 스타일

컨셉

카드 없음. 리스트 행만. iOS Settings/Apple Music 라이브러리 패턴을 차용한 극단적 간결함.

안티패턴 해결

  • 이모지 → Lucide SVG 20px 단색 아이콘
  • 파스텔 원 배경 → 제거 (아이콘만)
  • 36px 거대 숫자 → 12px pill 뱃지
  • 풀폭 CTA → 행 자체가 터치 타겟
  • 보라 그라디언트 헤더 → 텍스트 + pill
  • 그룹 이모지 → 텍스트만 (13px 회색)

장점

최소 시각적 소음. ADHD 스캔성 최고. 한눈에 전체 카테고리 파악 가능. 세로 공간 효율적.

단점

감성적 따뜻함 부족. DayStep의 Calm Luxe 느낌이 다소 희석될 수 있음. 너무 유틸리티 느낌.

컨셉

안티패턴 해결

  • 이중 장식 → 40px blue-50 원 + SVG (단일 장식)
  • 36px 숫자 → 13px 인라인 카운트 ("3건")

C: 진행률 요약 프로그레스 + 아코디언

컨셉

상단 세그먼트 프로그레스 바로 전체 구성 한눈에 파악. 아코디언으로 그룹 접기/펼치기. primaryColor 5단계 명도로 단색 톤 유지.

안티패턴 해결

  • 이모지 → Lucide SVG, 아코디언 헤더에는 컬러닷만
  • 이중 장식 → 컬러닷 10px (최소 시각 신호)
  • 36px 숫자 → 13px 비율 표시 (3/25)
  • 풀폭 CTA → 행 탭으로 바텀시트
  • 보라 헤더 → blue 계열 프로그레스 바
  • 그룹 이모지 → 컬러닷 + 텍스트

장점

진행감 시각화 — ADHD 동기부여에 효과적. 깔끔한 요약. 전체-부분 관계 직관적. 정보 밀도와 시각적 깔끔함 균형.

단점

인터랙션 단계 증가 (아코디언 열기 → 행 탭). "데이터 정리" 화면에 프로그레스 바가 다소 과잉일 수 있음.
