Design A: 통합 일정 패널 (TickTick 충실형)

TickTick과 동일한 검증된 UX 패턴 적용. 날짜+시간+알림+반복을 하나의 통합 패널로 관리합니다.

  • 📅 "오늘" 탭 → 키보드 사라지고 캘린더 패널 등장 (gap 없음)
  • 😀 아이콘 탭 → 키보드 유지 + 위에 아이콘 그리드 오버레이
  • 긴 컨텐츠(캘린더) → 키보드 숨김 / 짧은 컨텐츠(아이콘) → 키보드 유지
  • 아이콘을 제목 옆에서 툴바로 이동

Design B: 인라인 확장형 (패널 내부 확장)

날짜+시간+알림을 하나의 요약 칩으로 통합하여 정보 밀도를 높입니다. 패널이 아래서 위로 확장됩니다.

  • 통합 칩 탭 → 패널 확장되며 날짜/시간 카드 노출
  • 날짜/시간 카드를 탭하면 인라인 피커 등장
  • 확장 시 키보드 자연스럽게 내려감
  • 아이콘/우선순위는 짧으므로 키보드 위에 오버레이

Design C: 모드 전환형 (iOS 키보드 패턴)

하단 영역이 [키보드/일정/아이콘/우선순위] 모드를 전환합니다. iOS 이모지 키보드와 동일한 패턴.

  • 키보드와 설정 패널이 동일한 높이로 매끄러운 전환
  • ⌨️ 탭으로 언제든 키보드 복귀
  • 알림/반복은 인라인 칩으로 즉시 선택 (서브시트 없음)
  • 가장 빠른 워크플로우 (모드 전환만으로 모든 설정 접근)
