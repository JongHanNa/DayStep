# DayStep 앱 철학

## 핵심 정체성

DayStep은 단순한 할일 관리 앱이 아닌 **성인 ADHD 일상 관리 시스템**.
ADHD의 신경생물학적 어려움(과제 시작의 어려움, 동기 유지 불가, 시간 감각 왜곡, 작업 기억 결함, 자기 모니터링 부족, 감정 조절 어려움, 막연함으로 인한 회피)을 실질적으로 해결하여, 일상을 더 잘 돌보고 할일을 미루지 않게 돕는 것이 핵심 철학.

## 핵심 가치

**간단하게 · 바로 실행 · 함께 관리**

## 디자인 톤

자극적이지 않으면서 따뜻하고 격려적인, **"비난 없는 동반자"**

## ADHD 어려움 → 기능 매핑

| ADHD 어려움 | DayStep 기능 | 설명 |
|-------------|-------------|------|
| 과제 시작 어려움 | 집중 실행하기 | 타이머 + "지금 할 수 있는 가장 작은 것" 추천 |
| 동기 유지 불가 | 원동력 새기기 | "왜 해야 하는지" 기록 → 실행 전 상기 |
| 작업 기억 결함 | 관계 기록하기 | 대화/약속 즉시 기록 |
| 시간 감각 왜곡 | 일정 계획하기 + 소식 챙기기 | 시간축 시각화 |
| 자기 모니터링 부족 | 하루 계획하기 | 완료 시각화 → 자기효능감 |
| 감정 조절 어려움 | 감사 기록하기 | 긍정 경험 회상 |
| 막연함 회피 | AI 플래닝 | 큰 목표 → 실행 가능 단위 분해 |

## 설계 원칙

1. **낮은 진입장벽**: 시작하는 데 에너지가 거의 들지 않아야 한다
2. **즉각적 피드백**: 작은 행동에도 즉각적인 긍정 강화를 제공한다
3. **유연한 구조**: 완벽한 계획보다 "지금 할 수 있는 것"에 집중한다
4. **비판단적 환경**: 미완료에 대한 죄책감이 아닌, 완료에 대한 격려를 제공한다
5. **외부 기억 보조**: ADHD의 작업 기억 한계를 기술로 보완한다
