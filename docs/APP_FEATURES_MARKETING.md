## 한줄 소개

**DayStep — ADHD 친화적 일상 케어 시스템**

성인 ADHD의 실제 어려움(과제 시작, 동기 유지, 시간 감각, 작업 기억)을 과학적으로 이해하고, 할일·수면·청소·관계까지 일상 전체를 하나의 앱에서 돌볼 수 있는 올인원 시스템.

---

## 핵심 가치 제안 (Key Messages)

1. **ADHD 맞춤 설계** — 뇌과학 기반 기능 매핑 (전두엽, 도파민 보상회로, 실행기능)
2. **올인원 일상 관리** — 할일 + 포모도로 + 수면 + 청소 + 관계 + AI 계획을 하나의 앱에서
3. **즉시 실행 가능** — 낮은 진입장벽, "지금 할 수 있는 가장 작은 것" 추천

## 성인 ADHD의 핵심 어려움과 DayStep의 솔루션 매핑

> 이 섹션은 마케팅 메시지의 핵심입니다. 성인 ADHD가 일상에서 겪는 구체적 고통과, DayStep이 그것을 어떻게 해결하는지를 뇌과학 근거와 함께 정리합니다.

### ADHD 뇌의 3가지 핵심 취약점


| 취약점            | 과학적 원인                                 | 일상에서 겪는 고통                                |
| -------------- | -------------------------------------- | ----------------------------------------- |
| **전두엽 기능 저하**  | 실행기능(계획, 감정조절, 충동억제)의 중심인 전두엽의 활성도가 낮음 | "머리로는 알지만 몸이 안 움직인다", 계획→실행 전환 불가         |
| **도파민 기저치 부족** | 즉각적 보상에만 반응, 지연 보상에 둔감                 | 재미없는 일 시작 불가, 밤늦게 자극 추구 (SNS, 게임), 할일 미루기 |
| **작업 기억 결함**   | 전두엽의 작업 기억 용량 제한                       | "다음에 뭘 해야 하지?" 루프, 의사결정 피로, 멍해짐           |


### 어려움별 → DayStep 솔루션 상세

#### 1. 실행 마비 (Execution Paralysis)

> "하면 되잖아"라는 말을 ADHD인은 수없이 들어왔습니다. 하지만 이것은 '귀찮음'이 아니라 **'실행 마비'**입니다. 신경전달물질(도파민, 노르에피네프린)이 부족하면, 뇌가 의도를 행동으로 전환하는 경로 자체가 차단됩니다. **의지력의 문제가 아니라, 신경회로의 물리적 한계**입니다.


| DayStep 기능               | 해결 원리                                                         |
| ------------------------ | ------------------------------------------------------------- |
| **마이크로태스크 분해** (청소)      | "방 청소하기"가 아닌 "싱크대 그릇 3개 세척" — 시작 장벽을 극단적으로 낮춰 전두엽의 의사결정 부담 제거 |
| **"첫 단계" 안내** (청소 추천 카드) | "먼저: 새 시트 꺼내서 침대 위에 놓기" — 가장 작은 첫 행동을 제시해 실행 진입점 제공           |
| **포모도로 타이머** (실행 탭)      | "20분만 집중" — 무한한 과제를 유한한 시간 블록으로 전환해 시작 부담 감소                  |
| **빠른 집중 모드**             | 할일을 정하지 않고 바로 타이머 시작 가능 — 계획 단계 자체를 스킵                        |


#### 2. 동기 유지 불가 (Motivation Deficit)

> ADHD 뇌는 도파민 기저치가 일반인보다 낮습니다. "깨끗해진 방"이라는 미래 보상보다 "지금 당장 재미있는 것"에 끌릴 수밖에 없고, "30분 열심히 치워도 '아직도 이만큼 남았네'"라는 생각이 보상을 상쇄합니다.


| DayStep 기능             | 해결 원리                                              |
| ---------------------- | -------------------------------------------------- |
| **2~5분 단위 완료 체크** (청소) | ADHD 뇌는 도파민 보상 주기가 짧아야 동기 유지 — 매 완료마다 작은 도파민 히트 제공 |
| **체크보드 시각화** (청소)      | 칸을 채워가는 시각적 성취감 — 즉각적 보상 메커니즘                      |
| **연속 성공 스트릭** (수면 정원)  | "N일 연속 성공" — 스트릭 유지 동기                             |
| **원동력 새기기** (연료 탭)     | "왜 해야 하는지" 감정과 함께 기록 → 동기 약해질 때 상기                 |
| **진행률 원형 링** (홈)       | 오늘의 완료/전체를 한눈에 — 작은 성취도 시각화                        |


#### 3. 시간 감각 왜곡 (Time Blindness)

> ADHD인은 시간 흐름을 체감하기 어렵습니다. "조금만 더"가 2시간이 되고, "내일 하자"가 한 달이 됩니다.


| DayStep 기능               | 해결 원리                                             |
| ------------------------ | ------------------------------------------------- |
| **타임블록킹 뷰** (플래너 주/3일/일) | 할일을 시간 그리드에 배치 — 하루를 시각적으로 구조화                    |
| **예상 소요시간 표시** (청소)      | "예상 10분 ⚙3" — 각 태스크에 시간 감각 제공                     |
| **현재 기간 상기** (플래너 리플렉션)  | "지금 어떤 기간인가요? (시험, 프로젝트 마감)" — 거시적 시간 인식 보조       |
| **수면 시간 설정** (수면 정원)     | "🌙 11:30 PM → ☀️ 7:00 AM 목표 7시간 30분" — 수면 시간 구조화 |


#### 4. 의사결정 피로 (Decision Fatigue)

> "뭘 해야 하지?" → "어디부터?" → "언제 하지?" → 피로 → 아무것도 못 함. ADHD 뇌에서 이 의사결정 사이클은 전두엽 자원을 빠르게 소진시킵니다.


| DayStep 기능            | 해결 원리                                                 |
| --------------------- | ----------------------------------------------------- |
| **요일별 구역 자동 배정** (청소) | "오늘 뭘 해야 하지?"라는 의사결정 자체를 제거 — 앱을 열면 오늘 할 것이 이미 정해져 있음 |
| **아이젠하워 매트릭스** (플래너)  | 중요/긴급 4분면 — 우선순위 시각화로 판단 부담 감소                        |
| **AI 계획하기**           | "큰 목표를 실행 가능 단위로 분해" — AI가 분해 작업을 대행                  |
| **카테고리 자동 분류** (청소)   | 공간/디지털/소지품 3탭 — 이미 분류되어 있어 선택만 하면 됨                   |


#### 5. 에너지 변동성 (Energy Variability)

> ADHD인의 에너지는 날마다 크게 변동합니다. 어제는 방 전체를 치울 수 있었는데, 오늘은 컵 하나 치우는 것도 힘들 수 있습니다. 고정된 루틴은 "나쁜 날"에 실패 경험을 쌓고, 이 실패가 다음 시도의 장벽을 더 높입니다.


| DayStep 기능                | 해결 원리                                                  |
| ------------------------- | ------------------------------------------------------ |
| **3단계 에너지 레벨** (청소)       | 😊 괜찮음 / 📦 보통 / 😰 힘듦 — 오늘 컨디션에 맞는 태스크 양 자동 조절        |
| **과부하 방지**                | 힘든 날에는 2~3개만, 좋은 날에는 더 많이 — "이 정도는 할 수 있겠다"는 자기 효능감 생성 |
| **"하기 싫어도 해야 할 일"** (플래너) | 별도 섹션으로 분리 — "작은 용기가 큰 성장을 만들어요" 격려와 함께                |


#### 6. 수면 악순환 (Sleep Vicious Cycle)

> ADHD 뇌는 밤에도 과도하게 활성화됩니다. 피곤할수록 오히려 더 자극을 찾게 되어, 수면 부족 → 전두엽 기능 저하 → 충동 조절 실패 → 밤늦게 자극 추구 → 다시 수면 부족의 악순환이 **스스로 끊기지 않습니다**. 일반인은 100에서 80으로 떨어지지만, ADHD인은 70에서 50으로 떨어집니다.


| DayStep 기능      | 해결 원리                                              |
| --------------- | -------------------------------------------------- |
| **수면 정원**       | 수면 추적 + 나무 키우기 게이미피케이션 — 즉각적 보상 부재 문제를 시각적 성장으로 보완 |
| **스크린타임 연동**    | 수면 중 앱 사용 제한 — 자극 추구 경로 자체를 차단                     |
| **수면과 ADHD 교육** | 도파민 빚 메커니즘, 전두엽 이중 타격 등 과학적 설명 → 자기 이해 기반 행동 변화 유도 |


### 🔑 핵심 차별화: "자기 지각(Self-Awareness)" 기반 행동 변화

DayStep의 가장 독특한 차별점은 **ADHD 교육 콘텐츠가 기능 화면과 직접 연결**되어 있다는 것입니다.


| 기능 화면     | 연결된 교육 콘텐츠               | 효과                                                                                                                                |
| --------- | ------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| 수면 정원     | → "수면과 ADHD"             | 도파민 빚 메커니즘, 전두엽 이중 타격, 악순환 구조를 이해 → **"내가 밤에 폰을 놓지 못하는 건 의지 부족이 아니라 뇌의 도파민 보충 행동"**이라는 자기 지각 → 경각심 + 자기 연민 동시 발생 → 의식적 수면 행동 변화   |
| 청소/정리     | → "청소/정리와 ADHD"          | 실행기능 장애·도파민 부재·감각 과부하의 삼중 장벽 이해 → **"내가 게으른 게 아니라, 뇌가 작동하는 방식이 다른 것"**이라는 자기 수용 → 죄책감 감소 + 시스템 신뢰 → 앱이 제시하는 마이크로태스크에 저항 없이 따라가게 됨 |
| ADHD 이해하기 | 전두엽·도파민·실행기능·수면·휴식·신경다양성 | **자기 뇌의 작동 원리를 이해** → 자기 비난("왜 나만 이래") → 자기 이해("뇌가 이렇게 작동하니까") 전환 → 앱의 각 기능을 "내 뇌에 맞는 도구"로 인식                                     |


**이 "이해 → 지각 → 행동 변화" 사이클이 DayStep의 핵심 전략입니다:**

1. **교육 콘텐츠**로 자신의 ADHD 뇌를 과학적으로 이해시킴
2. **자기 지각**이 생겨 "이건 내 문제가 아니라 뇌의 특성"이라는 인식 전환
3. **경각심과 자기 연민**이 동시에 생겨 "의식적으로 더 제때 자려고" "더 청소하려고" 하는 내적 동기 발생
4. 앱의 **시스템(마이크로태스크, 에너지 레벨, 타이머)**을 "나를 돕는 도구"로 신뢰하고 따라감
5. **작은 성공 경험** 축적 → 자기 효능감 상승 → 지속적 사용

> 일반 할일 앱: "할 일을 관리하세요" (what)
> DayStep: "당신의 뇌가 왜 이렇게 작동하는지 이해하고, 그 뇌에 맞는 방식으로 일상을 돌보세요" (why + how)

## 화면별 상세

### 1. 홈 대시보드

**구조**: 2페이지 가로 캐러셀 + 세로 스크롤

> **ADHD 관점**: ADHD 뇌는 "오늘 뭘 해야 하지?"라는 질문 자체에서 전두엽 자원을 소진합니다. 홈 대시보드는 **앱을 여는 순간 오늘의 상황을 한눈에 보여주고, 다음 행동을 즉시 선택할 수 있는 허브** 역할을 합니다. "일상 돌보기 → 계획 세우기 → 생각과 기억"의 그룹 구조는 ADHD인의 삶에서 가장 무너지기 쉬운 3개 영역(자기 돌봄, 실행, 내면 관리)을 체계적으로 커버합니다.

#### 페이지 1 — 오늘의 현황


| 요소            | 설명                                                | ADHD 솔루션                                                  |
| ------------- | ------------------------------------------------- | --------------------------------------------------------- |
| 시간대 인사        | "좋은 오후예요 🌿" (아침/오후/저녁 자동 전환)                     | **비판단적 환경** — 몇 시에 열든 따뜻하게 맞이해 죄책감 없이 시작                  |
| 오늘의 진행률       | 원형 프로그레스 (완료/전체)                                  | **자기 모니터링 부족 대응** — 뇌가 하지 못하는 "지금 어디쯤인지" 추적을 시각적으로 대행     |
| 오늘의 미션        | 할일 미리보기 + "오늘 계획 세우기" CTA                         | **실행 마비 대응** — "뭘 해야 하지?" 단계를 스킵하고 원탭으로 계획 모드 진입          |
| **일상 돌보기** 그룹 | 수면 정원 🌙 · ADHD 이해하기 🧠 · 청소/정리하기 ○               | **자기 돌봄 사각지대 커버** — ADHD인이 가장 방치하기 쉬운 수면·청소·자기이해를 최상위에 배치 |
| **계획 세우기** 그룹 | 하루 계획하기 · 내 계획 보기 · AI로 계획하기 · Claude 연결 · 데이터 정리 | **실행기능 외부화** — 전두엽이 약한 계획·분류·구조화를 앱과 AI가 대행               |
| **생각과 기억** 그룹 | 원동력 새기기 💡 · 관계 기록하기 📈                           | **작업 기억 보조** — 금방 잊히는 동기와 대화를 외부 기억 장치에 저장                |


#### 페이지 2 — 영감 페이지


| 요소      | 설명                          | ADHD 솔루션                                                 |
| ------- | --------------------------- | -------------------------------------------------------- |
| 나의 원동력  | Notes에서 기록한 원동력 카드 (접기/펼치기) | **동기 유지 불가 대응** — 도파민이 바닥일 때 과거의 "왜 해야 하는지"를 즉시 상기       |
| 연락할 사람  | 소중한 사람 등록 → 연락 리마인더         | **작업 기억 결함 대응** — "연락해야지" 의도가 행동으로 전환되지 못하는 문제를 리마인더로 보완 |
| 오늘의 한 줄 | 매일 바뀌는 격려 문구                | **감정 조절 보조** — 자기 비난 루프에 빠지기 쉬운 ADHD인에게 외부 긍정 자극 제공      |

### 2. 플래너

**구조**: 2페이지 가로 캐러셀 + 세로 스크롤

> **ADHD 관점**: ADHD의 핵심 결함인 **실행기능(계획→우선순위→시간배치→실행→모니터링)** 전 과정을 외부 시스템으로 보조합니다. 특히 5가지 뷰(목록/월/주/3일/일)는 ADHD인의 **시간 감각 왜곡**(time blindness)을 다양한 시간 단위로 시각화해 "시간이 어떻게 흘러가는지"를 직관적으로 이해시킵니다. 2페이지의 일일 리플렉션은 ADHD 뇌가 자연적으로 하지 못하는 **자기 모니터링과 메타인지**를 구조화된 질문으로 유도합니다.

#### 페이지 1 — 할일 리스트


| 요소        | 설명                                                   | ADHD 솔루션                                       |
| --------- | ---------------------------------------------------- | ---------------------------------------------- |
| 주간 캘린더 헤더 | 월~토, 오늘 파란 하이라이트                                     | **시간 감각 보조** — "오늘이 한 주에서 어디쯤인지" 시각적 위치 인식     |
| 뷰 전환 메뉴   | **목록** · **월**(Pro) · **주** · **3일** · **일** — 5가지 뷰 | **시간 단위 유연성** — 거시(월) → 미시(일) 시간 감각을 상황에 맞게 전환 |
| 할일 리스트    | 빈 상태: "이 날짜에 할일이 없어요 + 버튼으로 추가해보세요"                  | **낮은 진입장벽** — 비어있어도 비난 없이 다음 행동을 안내            |
| + 플로팅 버튼  | → 할일 추가 바텀시트                                         | **즉시 외부화** — 떠오른 순간 바로 기록, 작업 기억에 의존하지 않음      |
| "오늘" 버튼   | 우상단, 오늘 날짜로 바로 이동                                    | **현재 시점 앵커링** — ADHD의 "지금"으로 즉시 복귀             |

| 우선순위 매트릭스     | 아이젠하워 4분면 (중요O/X × 긴급O/X)      | **의사결정 피로 감소** — "뭐부터 해야 하지?" 판단을 시각적 프레임워크로 단순화. ADHD 뇌가 긴급한 것에만 반응하는 패턴을 교정 |
| 하기 싫어도 해야 할 일 | 전용 섹션 + "작은 용기가 큰 성장을 만들어요"    | **회피행동 직면** — 도파민 보상이 없어 무한히 미루는 과제를 별도 공간에 명시. 격려 메시지로 실행 마비의 시작 장벽을 낮춤      |

| 오늘의 다짐/결단     | 자유 텍스트 입력                      | **의도의 외부화** — 전두엽이 유지하지 못하는 의도를 글로 고정시켜 실행 확률 상승                              |
| 오늘의 점검/회고/교훈  | 하루 마무리 성찰                      | **메타인지 훈련** — ADHD 뇌가 자연적으로 하지 못하는 "오늘 어땠는지" 돌아보기를 구조화된 질문으로 유도               |

### 3. 실행 (포모도로)

> **ADHD 관점**: ADHD의 가장 고통스러운 증상은 **"해야 하는 걸 알지만 시작할 수 없는"** 실행 마비입니다. 이것은 의지력 부족이 아니라, 도파민·노르에피네프린 부족으로 **의도→행동 전환 신경경로가 물리적으로 차단**되는 현상입니다. 포모도로 타이머는 "무한한 과제"를 "20분만"이라는 유한한 단위로 변환해 시작 장벽을 극적으로 낮춥니다. 특히 "빠른 집중" 모드는 **할일을 정하는 계획 단계 자체를 스킵**할 수 있어, 계획→실행 전환이 불가능한 순간에도 "일단 타이머만 누르기"가 가능합니다.


| 요소       | 설명                           | ADHD 솔루션                                       |
| -------- | ---------------------------- | ---------------------------------------------- |
| 타이머 링    | 큰 원형 타이머 **20:00** — "자유 모드" | **시간 시각화** — 추상적 시간을 줄어드는 원으로 구체화. 시간 감각 왜곡 보정 |
| 집중 시작 버튼 | 보라색 CTA                      | **원탭 실행** — 시작까지의 단계를 최소화해 실행 마비의 진입 장벽 제거     |
| 빠른 집중 링크 | 상단 파란 텍스트 → 포커스 선택 모달        | **계획 단계 스킵** — "뭘 할지"조차 정할 에너지가 없을 때 바로 시작 가능  |
| 포커스 통계   | 좌상단 🕐 아이콘 → 통계 모달           | **누적 성취 시각화** — ADHD 뇌가 무시하는 "작은 성취들"을 숫자로 증명  |

| --------- | ---------------------- | ------------------------------------------------- |

### 4. 연료 (원동력 노트)

> **ADHD 관점**: ADHD 뇌는 **도파민 기저치가 낮아 "하고 싶다"는 마음 자체가 잘 생기지 않습니다**. 지연된 보상에 둔감하기 때문에 "왜 이걸 해야 하지?"라는 동기가 유지되지 않습니다. 원동력 노트는 동기가 살아있는 순간에 감정과 함께 "왜"를 기록해두고, 도파민이 바닥난 순간에 다시 꺼내볼 수 있는 **외부 동기 저장소**입니다. 4가지 감정 칩(기쁨/감사/각성/결단)은 감정과 동기를 연결해, 나중에 읽었을 때 단순한 텍스트가 아닌 **감정적 회상**을 유도합니다.


| 요소       | 설명                      | ADHD 솔루션                                           |
| -------- | ----------------------- | -------------------------------------------------- |
| 타임라인 뷰   | 원동력 기록이 시간순으로 쌓임        | **동기의 아카이브화** — 순간적으로 타오르다 사라지는 ADHD의 동기를 물리적으로 보존 |
| + 플로팅 버튼 | → 원동력 새기기 바텀시트          | **감정이 살아있을 때 즉시 포착** — ADHD의 짧은 감정 주기에 맞춘 빠른 기록    |
| 빈 상태     | "첫 원동력을 작성해보세요" + 격려 문구 | **비판단적 온보딩** — "아직 없어도 괜찮아요"의 톤                    |


#### 원동력 새기기 바텀시트

- **4가지 감정 칩**: ♡ 기쁨 · ❤️ 감사 · ⚡ 각성 · ⚙️ 결단
  - → 감정 태깅으로 나중에 읽었을 때 **정서적 공명** 유도 (단순 텍스트 vs 감정 회상)
- 제목 (선택) + 본문 ("무엇이 나를 움직이게 하나요?")
  - → 질문형 프롬프트로 구체적 동기 추출 유도
- 저장하기 버튼

### 5-2. 수면 정원

> **ADHD 관점**: ADHD인의 수면 문제는 단순한 불규칙이 아닙니다. 도파민 기저치가 낮은 뇌가 **밤에 자극을 붙잡아 도파민을 보충**하려 하고(SNS, 게임, 영상), **멜라토닌 분비가 지연**되어 수면 리듬 자체가 뒤로 밀립니다. 피곤할수록 오히려 더 자극을 찾는 악순환이 스스로 끊기지 않습니다. 일반인은 수면 부족 시 전두엽이 100→80으로 떨어지지만, ADHD인은 **70→50으로 떨어져** 다음 날 실행기능이 사실상 마비됩니다. 수면은 ADHD 뇌의 전두엽을 충전하는 가장 기본적인 방법이며, 수면 없이는 어떤 전략도 제 효과를 발휘하기 어렵습니다.
>
> **"수면과 ADHD" 교육 화면** (우상단 메뉴에서 접근): 도파민 빚 메커니즘, 전두엽 이중 타격, 일반인 vs ADHD 비교표, 악순환 구조, 실행 마비의 신경과학적 설명 등을 제공하여 **"밤에 폰을 놓지 못하는 건 의지 부족이 아니라 뇌의 도파민 보충 행동"**이라는 자기 지각을 형성 → 경각심과 자기 연민이 동시에 생겨 의식적 수면 행동 변화를 유도합니다.


| 요소        | 설명                                          | ADHD 솔루션                                                    |
| --------- | ------------------------------------------- | ----------------------------------------------------------- |
| 연속 성공 배지  | "N일 연속 성공" 스트릭                              | **즉각 보상 부재 보완** — 수면은 즉각적 도파민 보상이 없는 활동. 스트릭으로 매일 작은 성취감 제공 |
| 뷰 전환 탭    | 일 · 주 · 월 · 년                               | **패턴 자기 인식** — ADHD 뇌가 감지하지 못하는 장기 수면 패턴을 시각화               |
| 수면 나무     | 수면 시간에 따라 나무가 성장 (🟢7h+ 🟡5-7h ⭐2-5h 🟠~2h) | **게이미피케이션** — 추상적인 "잘 잤다"를 구체적 시각 보상으로 전환                   |
| 수면 시간 설정  | 🌙 11:30 PM → ☀️ 7:00 AM + 목표 시간 배지         | **외부 구조 제공** — ADHD 뇌의 약한 내부 시간 감각을 대체하는 명확한 프레임            |
| 스크린타임 연동  | "수면 중 앱 사용 제한" 모달                           | **자극 추구 경로 차단** — 충동 조절이 어려운 ADHD 뇌를 위해 환경 자체를 변경           |
| 잠들기 시작 버튼 | 우하단 🌙 플로팅 버튼                               | **의식적 전환 의례** — "잠들기 시작"이라는 능동적 행위로 수면을 의식적 선택으로 전환         |
| 허용 앱 관리   | 수면 중 사용 가능한 앱 설정                            | **맞춤형 제한** — 모든 앱을 차단하면 저항감이 생기므로 허용 앱을 선택할 수 있게 함          |

### 5-3. 청소/정리하기

> **ADHD 관점**: ADHD 뇌에게 청소는 **실행기능 장애 + 도파민 보상 부재 + 감각 과부하**의 삼중 장벽입니다. "방 좀 치워야지"라는 단순한 의도조차 행동으로 전환되지 않는 이유는: (1) 어디부터 시작할지 결정·분류·순서 정하기가 모두 **전두엽의 작업 기억과 의사결정**에 의존하고, (2) "깨끗해진 방"이라는 미래 보상은 도파민 기저치가 낮은 뇌에 동기를 주지 못하며, (3) 어질러진 공간의 시각적 정보가 **감각 과부하**를 일으켜 뇌가 행동을 멈추게 하기 때문입니다. DayStep은 이 삼중 장벽을 **마이크로태스크 분해 + 에너지 적응형 + 루틴 자동화**로 무력화합니다.
>
> **"청소/정리와 ADHD" 교육 화면** (좌상단 ≡ 메뉴에서 접근): 실행기능 장애·도파민 부재·감각 과부하의 삼중 장벽을 과학적으로 설명하여 **"내가 게으른 게 아니라, 뇌가 작동하는 방식이 다른 것"**이라는 자기 수용을 형성 → 죄책감 감소 + 앱이 제시하는 마이크로태스크에 저항 없이 따라가게 됩니다.


| 요소          | 설명                             | ADHD 솔루션                                                                                                      |
| ----------- | ------------------------------ | ------------------------------------------------------------------------------------------------------------- |
| 에너지 레벨 선택   | 😊 괜찮음 · 📦 보통 · 😰 힘듦         | **에너지 변동성 대응** — ADHD인의 에너지는 매일 크게 변동. 고정 루틴은 "나쁜 날"에 실패→다음 장벽 상승의 악순환. 에너지에 맞는 양만 제안해 **과부하 방지 + 자기 효능감 유지** |
| 추천 태스크 카드   | 장소+작업명+첫 단계+예상 시간+난이도          | **시작 장벽 극적 해소** — "방 청소"가 아닌 "싱크대 그릇 3개 세척". 첫 단계("먼저: 새 시트 꺼내서 침대 위에 놓기")까지 안내해 의사결정 부담 제로                   |
| ▶ 시작 / ✅ 완료 | 파란/초록 버튼                       | **2~5분 도파민 사이클** — ADHD 뇌는 짧은 보상 주기가 필수. 매 완료마다 작은 도파민 히트 → 다음 태스크로 자연스러운 이행                                  |
| 체크보드 일러스트   | 완료 태스크 시각화 (칸 채우기)             | **즉각적 시각 보상** — "30분 치워도 아직 이만큼"이라는 좌절 대신, 채워지는 칸으로 진전을 가시화                                                   |
| 시간 범례       | 🟢10m+ · 🟡5-10m · ⭐2-5m · ~2m | **시간 감각 보조** — ADHD의 시간 왜곡 보정. "이거 5분이면 끝나"는 구체적 시간 감각 제공                                                     |
| 카테고리        | 🟢공간 · 🟣디지털 · 🟣소지품           | **인지 부하 감소** — 이미 분류되어 있어 "뭘 치워야 하지?" 의사결정 불필요                                                                |
| 요일별 자동 배정   | 오늘(수요일): 침실, 파일, 수납            | **의사결정 완전 제거** — "오늘 뭘 해야 하지?" 자체가 사라짐. 앱을 열면 이미 정해져 있음                                                       |
| 일일 태스크 리스트  | 장소별 체크리스트 (침실·주방·욕실·파일)        | **자동 조종 모드** — 다음에 뭘 할지 정할 필요 없이 순서대로 진행. 작업 기억 부담 제로                                                         |
| 뷰 전환        | 일 · 주 · 월 · 년                  | **장기 습관 추적** — ADHD 뇌가 인식하지 못하는 "나 이만큼 해왔다"를 시각적으로 증명                                                         |


**태스크 예시** (마이크로태스크 분해 원칙 적용):

- 침실: 먼지 제거 (주간, 5분), 옷장 정리, 이불 정리, 바닥 옷 수거
- 주방: 싱크대 닦기 (매일, 3분), 설거지 (매일, 10분)
- 욕실: 세면대 닦기 (매일, 2분)
- 파일: 다운로드 폴더 정리, 바탕화면 파일 정리

> 핵심 원리: "청소는 의지력 싸움이 아니라, ADHD 뇌에 맞는 시스템의 문제"

### 5-4. ADHD 이해하기

> **ADHD 관점 — 이 화면이 존재하는 이유**: 대부분의 생산성 앱은 "무엇을 하세요(what)"만 말합니다. DayStep은 **"당신의 뇌가 왜 이렇게 작동하는지(why)"**를 먼저 설명합니다. 이것이 핵심입니다: **자기 이해가 없으면 어떤 시스템도 "또 다른 실패 도구"**가 됩니다. ADHD인이 "왜 나만 이렇게 못하지?"라는 자기 비난에서 "내 뇌가 이렇게 작동하니까, 이 도구가 나를 도울 수 있어"로 전환되는 것이 **앱 지속 사용의 전제 조건**입니다. 3가지 학습 모드(비주얼 맵/아코디언/카드)는 ADHD인의 다양한 학습 선호와 주의력 패턴에 맞춘 것입니다.


| 요소        | 설명                                            | ADHD 솔루션                                                 |
| --------- | --------------------------------------------- | -------------------------------------------------------- |
| 뇌과학 비주얼 맵 | 중앙 뇌 아이콘 + 5개 토픽 (대표 키워드, 신경다양성, 수면, 흥식, 집안일) | **직관적 자기 이해** — 텍스트보다 시각적 맵이 ADHD 뇌에 더 접근 가능             |
| 뷰 전환 메뉴   | 비주얼 맵 · 아코디언 가이드 · 카드 리더 (3가지 학습 모드)          | **주의력 패턴 맞춤** — 시각형/구조형/탐색형 선택 가능                        |
| 대표적인 키워드  | 전두엽, 도파민과 보상회로, 실행기능 설명                       | **핵심 자기 지각** — "ADHD는 주의력 부족이 아니라 뇌의 관리 시스템이 원활하지 않은 상태" |
| 수면 토픽     | 잠들지 못하는 이유, 밤과 아침의 악순환, 야간 각성 줄이기             | **수면 문제의 탈수치화** — "게으른 게 아니라 멜라토닌 분비가 지연되는 것"            |
| 집안일 토픽    | 실행이 막막해지는 순간, 뇌과학적 이유, 실행을 돕는 방법              | **실행 마비의 정당화** — "전두엽 기능 저하 + 도파민 시스템 문제"                |
| 휴식 토픽     | 왜 쉬기가 어려운지, 편안한 휴식 만들기                        | **휴식 죄책감 해소** — "가만히 있으면 불안한 건 뇌의 과잉 활성 때문"              |
| 신경다양성 토픽  | 신경다양성의 의미와 가치, 존중과 지원                         | **자기 수용** — "뇌의 차이는 결함이 아니라 사람다움의 또 다른 모습"               |


**이 교육 콘텐츠의 행동 변화 효과**:

```
자기 비난 ("왜 나만 이래")
    ↓ ADHD 이해하기 교육
자기 이해 ("내 뇌가 이렇게 작동하는구나")
    ↓
자기 연민 + 경각심 동시 발생
    ↓
의식적 행동 변화 ("의지가 아니라 시스템이 필요하구나")
    ↓
앱의 기능을 "내 뇌에 맞는 도구"로 신뢰
    ↓
작은 성공 경험 축적 → 자기 효능감 → 지속 사용
```
