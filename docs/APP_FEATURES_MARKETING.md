# DayStep 앱 기능 — 마케팅 레퍼런스 문서

> 이 문서는 App Store 스크린샷, 마케팅 카피, 프로모션 소재 제작을 위한 앱 기능 레퍼런스입니다.
> 시뮬레이터 실사 탐색 + 소스코드 분석 기반 (2026-03-25 작성)

---

## 한줄 소개

**DayStep — ADHD 친화적 일상 케어 시스템**

성인 ADHD의 실제 어려움(과제 시작, 동기 유지, 시간 감각, 작업 기억)을 과학적으로 이해하고, 할일·수면·청소·관계까지 일상 전체를 하나의 앱에서 돌볼 수 있는 올인원 시스템.

---

## 핵심 가치 제안 (Key Messages)

1. **"비난 없는 동반자"** — 미완료에 죄책감이 아닌, 완료에 격려를 주는 따뜻한 톤
2. **ADHD 맞춤 설계** — 뇌과학 기반 기능 매핑 (전두엽, 도파민 보상회로, 실행기능)
3. **올인원 일상 관리** — 할일 + 포모도로 + 수면 + 청소 + 관계 + AI 계획을 하나의 앱에서
4. **즉시 실행 가능** — 낮은 진입장벽, "지금 할 수 있는 가장 작은 것" 추천

## 화면별 상세

### 1. 홈 대시보드

**구조**: 2페이지 가로 캐러셀 + 세로 스크롤

#### 페이지 1 — 오늘의 현황
| 요소 | 설명 | 마케팅 포인트 |
|------|------|-------------|
| 시간대 인사 | "좋은 오후예요 🌿" (아침/오후/저녁 자동 전환) | 따뜻한 첫인상 |
| 오늘의 진행률 | 원형 프로그레스 (완료/전체) | 한눈에 파악 |
| 오늘의 미션 | 할일 미리보기 + "오늘 계획 세우기" CTA | 즉시 실행 유도 |
| **일상 돌보기** 그룹 | 수면 정원 🌙 · ADHD 이해하기 🧠 · 청소/정리하기 ○ | 핵심 차별점 |
| **계획 세우기** 그룹 | 하루 계획하기 · 내 계획 보기 · AI로 계획하기 · Claude 연결 · 데이터 정리 | AI 연동 강조 |
| **생각과 기억** 그룹 | 원동력 새기기 💡 · 관계 기록하기 📈 | 감성적 기능 |

#### 페이지 2 — 영감 페이지
| 요소 | 설명 |
|------|------|
| 나의 원동력 | Notes에서 기록한 원동력 카드 (접기/펼치기) |
| 연락할 사람 | 소중한 사람 등록 → 연락 리마인더 |
| 오늘의 한 줄 | 매일 바뀌는 격려 문구 |

### 2. 플래너

**구조**: 2페이지 가로 캐러셀 + 세로 스크롤

#### 페이지 1 — 할일 리스트
| 요소 | 설명 |
|------|------|
| 주간 캘린더 헤더 | 월~토, 오늘 파란 하이라이트 |
| 뷰 전환 메뉴 | **목록** · **월** (Pro) · **주** · **3일** · **일** — 5가지 뷰 |
| 할일 리스트 | 빈 상태: "이 날짜에 할일이 없어요 + 버튼으로 추가해보세요" |
| + 플로팅 버튼 | → 할일 추가 바텀시트 |
| "오늘" 버튼 | 우상단, 오늘 날짜로 바로 이동 |

#### 페이지 2 — 일일 리플렉션
| 요소 | 설명 | 마케팅 포인트 |
|------|------|-------------|
| 우선순위 매트릭스 | 아이젠하워 4분면 (중요O/X × 긴급O/X) | ADHD 우선순위 시각화 |
| 하기 싫어도 해야 할 일 | 전용 섹션 + "작은 용기가 큰 성장을 만들어요" | ADHD 회피행동 대응 |
| 현재 기간 상기 | "지금 어떤 기간인가요? (시험, 프로젝트 마감 등)" | 시간 감각 보조 |
| 오늘의 다짐/결단 | 자유 텍스트 입력 | 자기 동기부여 |
| 오늘의 점검/회고/교훈 | 하루 마무리 성찰 | 자기 모니터링 |

#### 할일 추가 바텀시트
- 제목 입력 + 설명 추가
- 날짜 칩 (📅 오늘)
- ⚙️ 카테고리/아이콘 선택 팝오버
  - **7개 카테고리**: 일상 · 업무 · 건강 · 이동 · 소통 · 학습 · 기타
  - 카테고리별 **8개 아이콘**

#### 뷰 모드 상세
| 뷰 | 무료/Pro | 설명 |
|----|---------|------|
| 목록 | 무료 | 날짜별 할일 리스트 |
| 월 | **Pro** | 월간 캘린더 + 날짜별 도트 + 드래그 이동 |
| 주 | 무료 | 7일 타임그리드 (시간별 블록 배치) |
| 3일 | 무료 | 3일 타임그리드 |
| 일 | 무료 | 1일 타임그리드 |

### 3. 실행 (포모도로)

| 요소 | 설명 | 마케팅 포인트 |
|------|------|-------------|
| 타이머 링 | 큰 원형 타이머 **20:00** — "자유 모드" | 미니멀한 집중 UI |
| 집중 시작 버튼 | 보라색 CTA | 원탭 시작 |
| 빠른 집중 링크 | 상단 파란 텍스트 → 포커스 선택 모달 | 즉시 실행 |
| 포커스 통계 | 좌상단 🕐 아이콘 → 통계 모달 | 성취감 시각화 |

#### 포커스 선택 모달
- **할일 집중**: 플래너의 할일 선택 → 해당 할일에 집중
- **빠른 집중**: 할일 없이 바로 시작 → 끝나면 기록

#### 포커스 통계 모달
| 메트릭 | 설명 |
|--------|------|
| 오늘의 포모 | 오늘 완료한 세션 수 |
| 오늘의 집중 | 오늘 집중 시간 (분) |
| 총 포모 | 전체 완료 세션 수 |
| 총 집중 기간 | 누적 집중 시간 |
| 연속 집중 현황 | 현재 연속 / 최고 연속 |

### 4. 연료 (원동력 노트)

| 요소 | 설명 | 마케팅 포인트 |
|------|------|-------------|
| 타임라인 뷰 | 원동력 기록이 시간순으로 쌓임 | 나만의 동기부여 아카이브 |
| + 플로팅 버튼 | → 원동력 새기기 바텀시트 | 간편한 기록 |
| 빈 상태 | "첫 원동력을 작성해보세요" + 격려 문구 | 온보딩 유도 |

#### 원동력 새기기 바텀시트
- **4가지 감정 칩**: ♡ 기쁨 · ❤️ 감사 · ⚡ 각성 · ⚙️ 결단
- 제목 (선택) + 본문 ("무엇이 나를 움직이게 하나요?")
- 저장하기 버튼

### 5-3. 청소/정리하기

| 요소 | 설명 | 마케팅 포인트 |
|------|------|-------------|
| 뷰 전환 | 일 · 주 · 월 · 년 | 청소 습관 추적 |
| 체크보드 일러스트 | 완료 태스크 시각화 (칸 채우기) | 성취감 게이미피케이션 |
| 에너지 레벨 선택 | 😊 괜찮음 · 📦 보통 · 😰 힘듦 | **ADHD 핵심: 에너지에 맞춘 추천** |
| 추천 태스크 카드 | 장소+작업명+첫 단계+예상 시간+난이도 | 마이크로 태스크 분해 |
| ▶ 시작 / ✅ 완료 | 파란/초록 버튼 | 즉시 실행 |
| 시간 범례 | 🟢10m+ · 🟡5-10m · ⭐2-5m · ~2m | 시간 감각 보조 |
| 카테고리 | 🟢공간 · 🟣디지털 · 🟣소지품 | 영역별 분류 |
| 일일 태스크 리스트 | 장소별 체크리스트 (침실·주방·욕실·파일) | 구조화된 루틴 |

**태스크 예시**:
- 침실: 먼지 제거 (주간, 5분), 옷장 정리, 이불 정리, 바닥 옷 수거
- 주방: 싱크대 닦기 (매일, 3분), 설거지 (매일, 10분)
- 욕실: 세면대 닦기 (매일, 2분)
- 파일: 다운로드 폴더 정리, 바탕화면 파일 정리

## 타겟 유저 페르소나

| 페르소나 | 핵심 니즈 | DayStep이 주는 가치 |
|----------|----------|-------------------|
| 성인 ADHD 진단자 | 일상 구조화, 실행력 보조 | 에너지 맞춤 추천, 마이크로 태스크, 포모도로 |
| ADHD 의심 자가관리자 | 자기 이해, 습관 형성 | ADHD 교육, 수면/청소 루틴, 원동력 기록 |
| 일상 관리 어려운 직장인 | 할일 관리, 시간 관리 | 아이젠하워 매트릭스, 타임블록킹, AI 계획 |
| 미루기 습관 개선 희망자 | 동기부여, 시작의 어려움 극복 | "하기 싫어도 해야 할 일" 섹션, 원동력 노트 |
