현재 원동력 새기기 화면 — 5가지 차원 분석

실제 코드 기반으로 분석한 현재 구현의 문제점

🎨

1. 시각적 스타일링

  • ● 오렌지 CTA 과도 — !bg-orange-500 btn-lg h-16 크기가 공격적, 화면 상단을 독점
  • ● 처리/미처리 구분 부족 — CheckCircle2 아이콘 하나로만 구분, 배경색 차이 없음
  • ● 단조로운 카드 — 모든 카드가 bg-base-200로 동일, 감정적 차별화 없음
  • ● 빈 상태 없음 — 원동력이 0개일 때 별도 디자인 없이 빈 화면만 표시
👆

2. 인터랙션 패턴

  • ● 드롭다운 숨김 — 핀 고정/삭제가 MoreHorizontal 드롭다운 안에 숨겨져 발견성 낮음
  • ● 모바일 스와이프 없음 — 터치 제스처 미지원, 탭만 가능
  • ● 할일 커스터마이징 불가 — 원동력에서 할일 생성 시 제목만 입력, 상세 설정 불가
  • ● 인라인 편집 없음 — 수정 시 반드시 모달 열어야 함
🗂️

3. 정보구조

  • ● 카테고리/태그 없음 — 모든 원동력이 단일 리스트, 감정/목적별 분류 불가
  • ● 검색/필터 없음 — 원동력이 많아지면 찾기 어려움
  • ● 시간순만 가능 — 미처리 우선, 생성일순 정렬만 제공
  • ● 배너 고정 숨김 — 홈 배너에 표시하는 핀 기능이 드롭다운 안에 있어 인지 어려움
🔗

4. 태스크 플로우

  • ● 원동력→할일 단절 — 작성 후 "할일 만들기" 클릭 필요, 자연스런 흐름 X
  • ● 진행 시각화 없음 — 연결된 할일 수만 배지로 표시, 완료율 미표시
  • ● 완료 연결 미표시 — 할일 완료 시 원동력 카드에 반영 없음
  • ● 3단 CTA 혼란 — 모달 내 "지금 시작/일정 잡기/저장하기" 3개 버튼의 차이 불명확
🧠

5. 행동 설계 (ADHD 친화성)

  • ● 게이미피케이션 없음 — 성취감/보상 메커니즘 부재, 지속 동기 약화
  • ● 리마인더 없음 — 작성한 원동력을 다시 보게 하는 메커니즘 X
  • ● 연속 기록 없음 — 꾸준한 원동력 작성 추적/시각화 없음
  • ● 수동적 메시지 — 배너에 표시할 원동력을 직접 선택해야, 랜덤/자동 추천 없음

🎯 감정 태그 필터

기쁨·감사·각성·결단 4가지 감정으로 분류. ADHD 사용자는 감정 기반 회상이 효과적 — 논리적 분류보다 감정 연결이 동기 회복에 직결.

📱 FAB로 CTA 교체

화면 상단의 공격적 오렌지 버튼 대신 우하단 FAB로 교체. 콘텐츠 영역 확보 + 엄지 접근성 최적화.

🔄 인라인 확장 카드

카드 클릭 시 모달 없이 아코디언 확장. 상세 내용 + 할일 진행률을 인라인으로 표시하여 컨텍스트 유지.

📊 할일 진행률 인라인 표시

연결된 할일의 완료율을 카드 내부에 프로그레스 바로 표시. 원동력→행동의 흐름을 시각적으로 연결.

🟢 상태 도트

카드 우상단 작은 도트로 처리 상태 표시 — 초록(할일 있음), 회색(미처리). 배경색 차이 없이도 즉시 인지.

🌡️ 그라데이션 헤더

amber→rose 따뜻한 그라데이션으로 감정적 톤 설정. 연속 기록·총 원동력·전환율 통계를 컴팩트하게 표시.

🏆 XP & 레벨 시스템

원동력 작성(+10), 할일 전환(+20), 할일 완료(+30)로 XP 획득. 레벨업 시 새 뱃지 해금. ADHD 사용자에게 즉각적 보상 피드백 제공.

🃏 카드 뒤집기 인터랙션

앞면(원동력 텍스트 + 감정 태그) ↔ 뒷면(연결 할일 + 진행률). 즐거운 인터랙션으로 탐색 유도, 정보 밀도를 자연스럽게 분리.

🔥 연속 기록 (Streak)

매일 원동력 작성 시 연속 기록 증가. 불꽃 아이콘과 일수 표시. 습관 형성을 위한 시각적 동기 부여.

🎯 주간 챌린지

매주 새로운 미니 챌린지 제공 (예: 매일 1개 작성, 3개 할일 전환). 단기 목표로 장기 습관 유도.

🏅 성취 뱃지

마일스톤 달성 시 뱃지 해금 — 첫 원동력(🌱), 7일 연속(🔥), 100% 전환(💎). 수집 욕구 자극으로 지속 사용 유도.

🎨 그라데이션 카드 색상

감정 태그별 고유 그라데이션 — 결단(보라), 감사(초록), 각성(주황). 시각적 다양성으로 단조로움 해소.

📝 상단 고정 인라인 입력

모달 없이 화면 상단에 항상 노출된 입력 바. 생각나는 즉시 작성 가능 — ADHD의 "지금 아니면 잊는다" 특성에 최적화.

🔍 필터/검색 바

전체·미처리·처리완료 3탭 필터 + 검색 아이콘. 항목이 많아져도 빠르게 원하는 원동력 찾기 가능.

🟢 상태 도트

초록(처리완료) / 회색(미처리)의 작은 도트로 한눈에 상태 파악. 최소한의 시각 요소로 최대 정보 전달.

👆 호버 퀵 액션

마우스 호버 시 📌(배너 고정) · ✚(할일 생성) · 🗑(삭제) 3개 버튼 즉시 노출. 드롭다운 메뉴 탐색 불필요.

✏️ 인라인 편집

텍스트 클릭 시 바로 편집 모드 진입, 포커스 아웃 시 자동 저장. 모달 전환의 컨텍스트 손실 방지.

📊 인라인 통계

각 항목 아래에 "할일 2개 · 50%" 형태로 진행 상태 즉시 표시. 별도 화면 전환 없이 전체 현황 파악.

💡 권장 방향

ADHD 사용자 최적: Demo B (게이미피케이션) — 즉각적 보상, 시각적 진행, 습관 형성 메커니즘이 ADHD의 동기 유지 결함에 가장 효과적.

최소 비용 개선: Demo C (미니멀) — 현재 구조에서 가장 적은 변경으로 핵심 문제(CTA, 상태 구분, 액션 접근성)를 해결.

하이브리드 제안: C의 인라인 입력 + A의 감정 태그 + B의 연속 기록/XP를 조합하면 실용성과 동기 부여를 동시에 달성.
