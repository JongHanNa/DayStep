# DayStep QA 테스트 체크리스트

> **목적**: 매 릴리즈(iOS/Android) 전 수동 검증용 체크리스트
> **대상**: iOS + Android 실기기 (양쪽 동일하게 수행)
> **최종 업데이트**: 2026-04-10

---

## 사용 방법

1. 릴리즈 빌드 설치 후, 아래 체크리스트를 **iOS / Android 각각** 수행
2. 각 항목 통과 시 `[x]` 체크, 실패 시 `[!]` 표시 + 증상 메모
3. **스모크 테스트** (⚡ 표시) → 모든 릴리즈에서 필수
4. **전체 회귀 테스트** → 메이저 업데이트 또는 대규모 변경 시

### 테스트 환경 매트릭스

| 구분    | iOS                  | Android                 |
| ------- | -------------------- | ----------------------- |
| 최신 OS | iPhone 15+ / iOS 17+ | Pixel 7+ / Android 14+  |
| 구형 OS | iPhone 11 / iOS 16   | Galaxy S10 / Android 12 |
| 태블릿  | iPad (선택)          | Galaxy Tab (선택)       |

## 14. 크로스 플랫폼 공통 검증

### 네트워크 상태

- [ ] ⚡ 오프라인 → 적절한 안내 메시지 (크래시 없음)
- [ ] 약한 네트워크 (3G) → 타임아웃 처리 정상
- [ ] 오프라인 → 온라인 복귀 → 데이터 동기화

### 앱 생명주기

- [ ] ⚡ 백그라운드 → 포그라운드 복귀 → 크래시 없음
- [ ] 앱 강제 종료 → 재시작 → 정상 동작
- [ ] 메모리 부족 → 앱 종료 → 재시작 시 데이터 유지
- [ ] 화면 회전 (지원 시) → 레이아웃 정상

### 시스템 인터럽트

- [ ] 전화 수신 중 → 앱 복귀 정상
- [ ] 알림 배너 표시 중 → 앱 동작 정상
- [ ] 제어센터/알림센터 열기 → 복귀 정상
- [ ] 스크린샷 촬영 → 크래시 없음

### 키보드 상호작용

- [ ] ⚡ 텍스트 입력 시 키보드 올라옴 → 입력 필드 가려지지 않음
- [ ] 키보드 외부 탭 → 키보드 닫힘
- [ ] 다음 필드 이동 (탭키/다음 버튼) 정상
- [ ] 이모지 입력 정상
- [ ] 긴 텍스트 입력 → 스크롤 가능

### 퍼포먼스 체감

- [ ] ⚡ 앱 시작 → 홈 화면 표시 3초 이내
- [ ] 탭 전환 → 0.5초 이내 화면 전환
- [ ] 스크롤 → 끊김 없이 60fps 유지
- [ ] 애니메이션 (호흡, 프로그레스 링) → 부드러운 동작
- [ ] 목록 100개+ 항목 → 버벅임 없음

### 접근성

- [ ] VoiceOver (iOS) / TalkBack (Android) → 주요 요소 읽힘
- [ ] 텍스트 크기 확대 (시스템 설정) → 레이아웃 깨지지 않음
- [ ] 고대비 모드 → 텍스트 읽을 수 있음

### 딥링크 / 푸시 알림

- [ ] 푸시 알림 탭 → 해당 화면 진입
- [ ] 알림 권한 거부 → 앱 정상 동작 (크래시 없음)

7. 노트 FAB → 동기부여 작성/저장
8. 설정 → 구독 상태 표시
9. 로그아웃 → 로그인 화면 복귀
10. 오프라인 → 크래시 없음
11. 키보드 → 입력 필드 가려지지 않음
12. 앱 시작 3초 이내, 탭 전환 부드러움
13. 인앱 구매 상품 표시
14. 앱 업데이트 → 데이터 유지
