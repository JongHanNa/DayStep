# 이력서 생성 요청 가이드

> 채용공고별 맞춤 이력서를 Claude에게 요청할 때 사용하는 템플릿. **이전 이력서를 항상 함께 첨부**하므로 프로필/구조/커스터마이징 가이드는 생략.

---

## 요청 시 제공할 정보 체크리스트

**필수**:
- [ ] 채용공고 URL 또는 텍스트 내용
- [ ] 참조할 기존 이력서 (기본: 가장 최근 이력서)

**선택**:
- [ ] 회사에 대해 알게 된 추가 정보
- [ ] 특별히 강조하고 싶은 경험/기술
- [ ] 지원동기에 녹이고 싶은 개인 이야기

---

## 요청 예시 템플릿

```
이 채용공고에 맞는 이력서를 만들어줘.
기존 이력서의 내용을 참고하고, 기존 이력서의 형식과 구조 안에서 맞춤형으로 새 이력서를 생성해줘.

[채용공고 URL 또는 내용 붙여넣기]

참고:
- 이 회사는 ○○ 분야에 특화된 스타트업이야
- Electron 경험을 특히 강조해줘
- 기존 이력서: docs/portfolio/이력서_제이엘티_나종한.md
```
