# ADHD 도움말 텍스트 보관

각 ADHD 모드의 탭바 도움말 텍스트를 보관합니다. 추후 UI 재사용 시 참고.

---

## FuelMode (원동력/머릿속정리)

| 탭 | ADHD 특성 | 기능 설명 |
|---|---|---|
| timeline (타임라인) | 자기 모니터링(Self-Monitoring) 결함. "내가 뭘 했지?" 파악이 어렵습니다. | 완료한 할일 시간순 시각화 → 작은 성취도 눈에 보임, 자기효능감 강화! |
| execute (실행) | 과제 시작의 어려움(Task Initiation). 해야 할 건 알지만 시작 버튼이 안 눌려요. | 타이머 + 방해차단 + 원동력 상기 → 시작의 마찰을 줄여 첫 발을 내딛도록 도움! |
| motivation (원동력) | 동기 유지 결함(Motivation Deficit). 중요한 건 알지만 하고 싶은 마음이 안 생겨요. | 왜 해야 하는지, 완료 후 기분을 미리 적어두고 → 실행 전 다시 보며 동기 충전! |
| organize (정리) | 조직화 결함. 머릿속이 복잡하고 할일이 뒤엉켜 어디서 시작할지 막막해요. | 미분류 할일만 모아서 표시 → 정리해야 할 것만 집중, 인지 부하 감소! |

## RelationshipInsightsMode (관계 인사이트)

| 탭 | ADHD 특성 | 기능 설명 |
|---|---|---|
| record (기록) | 작업기억(Working Memory) 결함으로 대화 내용, 약속, 부탁 등을 쉽게 잊어버립니다. "들었는데 기억이 안 나요" | 만남 직후 바로 기록 → 외부 기억 저장소 역할. 부탁/약속을 할일로 연동하여 잊어버림 방지! |
| news (소식) | 시간 감각 왜곡(Time Blindness)으로 "언제 연락했더라?" 추적이 어렵습니다. 관계 소홀 인식이 늦어요. | 기록된 소식을 시간순 조회 → 연락 빈도 시각화, 소홀한 관계 파악에 도움! |
| gratitude (감사) | 부정 편향(Negativity Bias) 강화. 감정 조절 어려움으로 좌절감에 쉽게 압도됩니다. | 감사한 순간 기록 → 긍정 경험 의도적 회상, 정서 균형 회복에 도움! |

## TaskOrganizeMode (할일 정리)

| 탭 | ADHD 특성 | 기능 설명 |
|---|---|---|
| timeline (타임라인) | 자기 모니터링(Self-Monitoring) 결함. "내가 뭘 했지?" 파악이 어렵습니다. 성취감 인식이 부족해요. | 완료한 할일 시간순 시각화 → 작은 성취도 눈에 보임, 자기효능감 강화! |
| organize (정리) | 조직화/우선순위 설정 결함. 할일이 쌓이면 어디서 시작할지 막막합니다. | 미분류 할일만 모아서 표시 → 정리해야 할 것만 집중, 인지 부하 감소! |
| stats (통계) | 피드백 민감도 저하. 자신의 패턴 인식이 어렵고, 같은 실수를 반복하게 됩니다. | 완료율, 시간대별 패턴 분석 → 데이터 기반 자기 이해, 최적 루틴 발견! |
| graph (그래프) | 전체 구조 파악의 어려움. 할일들 사이의 관계를 놓치기 쉽습니다. | 할일/프로젝트/목표 관계 시각화 → 전체 그림 파악, 우선순위 결정 도움! |

## ProjectMode (AI 계획)

**제목**: AI와 함께 계획하면 더 쉬워요

**설명**: 중요하고 긴급한데 시작이 어렵고 막연한 것들을 여기서 관리하세요.

**ADHD 특성**: 집행기능(executive function) 특성상, 중요한 일도 "어디서부터?"가 막연하면 미루게 됩니다.

**기능 설명**: DayStep MCP는 막막한 계획을 "폴더 열기 → 파일 1개 만들기 → ..."처럼 "뇌에 친절한 단위"로 쪼개줍니다.

**추가 메시지**: 작업 시작이 90% — 시작만 하면 나머지는 따라와요!
